# Contexto de Uso

O aplicativo é utilizado por pessoas em deslocamento urbano,
muitas vezes correndo para não perder ônibus ou metrô,
em ambientes externos, com luminosidade intensa,
ruído e necessidade de tomada rápida de decisão.

---

# Decisões de UX

Priorizamos:

- Tempo de chegada maior que número da linha
- Botões grandes para uso com apenas um polegar
- Pouca informação por tela
- Alto contraste visual
- Navegação simples e direta
- Redução de carga cognitiva

A lotação dos veículos foi destacada para auxiliar
na escolha rápida do usuário.

---

# Acessibilidade

O protótipo auxilia pessoas com mobilidade reduzida através de:

- Botões grandes
- Poucos toques necessários
- Informações objetivas
- Indicação de acessibilidade no veículo
- Fluxo simplificado
- Compatibilidade com uso em movimento

---

# Fluxo Principal

Busca de rota
→ Resultado
→ Mapa
→ Detalhes do veículo
→ Carteira digital

# Desafio Extra

## Modo Noturno
A interface reduziria brilho e utilizaria fundo escuro
para evitar ofuscamento.

## Offline
O sistema exibiria aviso de perda de conexão
e manteria último trajeto armazenado localmente.
